Haulio Bulk CSV Import Guide

Use this guide to import 300-500 loads quickly from a spreadsheet.

Overview

Export two CSV files from your spreadsheet and upload them in the UI (Admin or Dispatch). The importer creates loads, stops, and links them by loadNumber.

Required Files

  • loads.csv - one row per load
  • stops.csv - one row per stop, linked by loadNumber

loads.csv Columns

ColumnDescription
loadNumberUnique load ID (required)
customerNameCustomer name
statusPLANNED, ASSIGNED, IN_TRANSIT, DELIVERED, READY_TO_INVOICE, INVOICED
rateNumeric linehaul rate
shipperReferenceNumberOptional shipper reference number
consigneeReferenceNumberOptional consignee reference number
palletCountOptional pallet count (integer)
weightLbsOptional weight in lbs (integer)
assignedDriverEmailDriver email (must exist in system)
truckUnitTruck unit (auto-created if missing)
trailerUnitTrailer unit (auto-created if missing)

stops.csv Columns

ColumnDescription
loadNumberMatches loads.csv
sequenceOrder of stops (1, 2, 3...)
typePICKUP (Shipper) or DELIVERY (Consignee)
nameStop name (yard or customer)
addressStreet address
cityCity
stateState
zipZIP
appointmentStartISO date time (optional)
appointmentEndISO date time (optional)

Yard -> Yard -> Consignee Pattern

For LTL and cross-dock flows, use three stops:

  1. Origin Yard (PICKUP)
  2. Destination Yard (PICKUP)
  3. Final Consignee (DELIVERY)

UI Import Steps

  1. Go to /admin or /dispatch
  2. Open the Bulk load import panel
  3. Upload loads.csv and stops.csv
  4. Optional: check Wipe existing loads
  5. Click Import CSV

Example

loads.csv
loadNumber,customerName,status,rate,shipperReferenceNumber,consigneeReferenceNumber,palletCount,weightLbs,assignedDriverEmail,truckUnit,trailerUnit
LD-3001,Delta Foods,PLANNED,1450,SHIP-REF-1,CONS-REF-9,24,42000,driver@demo.com,T-101,TR-201

stops.csv
loadNumber,sequence,type,name,address,city,state,zip,appointmentStart,appointmentEnd
LD-3001,1,PICKUP,Origin Yard,100 Yard Way,Memphis,TN,38103,2026-02-01T14:00:00Z,2026-02-01T16:00:00Z
LD-3001,2,PICKUP,Destination Yard,400 Yard Ave,Nashville,TN,37209,2026-02-01T20:00:00Z,2026-02-01T22:00:00Z
LD-3001,3,DELIVERY,Final Consignee,88 Market St,Nashville,TN,37209,2026-02-02T02:00:00Z,2026-02-02T04:00:00Z

Notes

  • loadNumber must match between files.
  • sequence must be unique per load.
  • Date fields should be ISO-8601 (UTC) for best results.
